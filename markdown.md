# 🥊 Vanilla JS Fighting Game (纯JS格斗游戏)

只用原生 HTML/CSS/JavaScript 手搓的 2D 格斗游戏，不依赖任何第三方库和框架。
这是为了锻炼底层逻辑而编写的 MVP（最小可行性产品）版本。

## ✨ 已实现的核心功能
- **状态机输入**：使用按键状态表解决了按键冲突，实现流畅的双向移动与跳跃。
- **手搓物理引擎**：模拟重力加速度与下落加速，实现真实的跳跃抛物线。
- **AABB碰撞检测**：纯数学计算矩形重叠，实现攻击判定。
- **战斗手感调优**：
  - 🥊 **击退**：基于动量守恒的模拟，配合摩擦力指数衰减，实现自然的滑行。
  - ⏱️ **顿帧**：命中瞬间暂停世界时间 60ms，极大增强打击感。
  - 💥 **反冲**：攻击者命中时的微小后退，强化双向交互感。
- **基础敌人 AI**：视野追踪 + 自动挥拳判定。

## 🛠️ 技术亮点
- 拒绝调库，纯 `requestAnimationFrame` 驱动游戏主循环。
- DOM 操作与 CSS Transition 结合，实现丝滑的状态切换（攻击、受击变色）。
